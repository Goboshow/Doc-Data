Requirements:
        - Python 3.x installed (Python 3.10+ recommended)
        - "tkinter" (included with standard python installations)
        - A local computer (cannot run in cloud IDEs, like GitHub Codespaces, as I'm too tired to figure it out)
        - The FABULABULOUS png must be in the EXACT same folder as "maincode.py", no more, no less.


How to Run:
        - First, start moving your legs- Kidding. Download or clone the repository.
        - Make sure "maincode.py" and the fabulabulous image are in the same folder.
        - Open your terminal / command prompt and navigate to that folder, for example:
                                                                                        cd "C:\Users\Ben\PaintDoctor"
        - Run the program with "python main.py", or if multiple versions are installed, "python3 main.py".
